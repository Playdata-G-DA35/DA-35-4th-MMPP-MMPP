# MMPP (Make My Profile Photo)
  
## 1. 프로젝트 주제 
  **MMPP(Make My Profile Photo)**

      특정 아트 스타일을 기반으로 사용자의 얼굴 사진을 재구성한 프로필 사진을 생성하여 제공한다.

## 2. 팀원 및 역할
 - 고서영 : 모델 테스트 및 데이터셋 수집
 - 김기현 : 문서 작업 관리 및 데이터셋 수집
 - 이서희 : 모델 탐색 및 구현
 - 이주호 : 구현 환경 조사 및 데이터셋 수집
